# Application Monitoring Dashboard

## Overview
This project is a Python-based Application Monitoring Dashboard built using Streamlit and Pandas. It helps monitor application performance metrics and visualize operational KPIs.

## Features
- Displays system log data
- Tracks average response time
- Monitors CPU usage
- Tracks application errors
- Interactive dashboard visualization

## Technologies Used
- Python
- Streamlit
- Pandas

## Installation

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
